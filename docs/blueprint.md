# **App Name**: ProcessFit Analyzer

## Core Features:

- Questionnaire UI: Present a dynamic questionnaire with various input types (text, number, dropdown, radio) to collect process data.
- Data Submission: Allow users to submit their questionnaire responses for analysis.
- Scoring Engine: Calculate scores for volume, cost, risk, feasibility, and strategic impact based on user inputs, including penalty applications.
- Process Categorization: Categorize the process as 'Quick Win', 'Strategic Long-Term', 'Incremental Gains', or 'Avoid/Revisit' based on the calculated scores.
- AI-Powered Analysis: Submit the questionnaire responses and calculated scores to the Gemini LLM and a tool will provide a recommendation, confidence level, top strengths, concerns, a timeline estimate, and suggested next steps.
- Results Dashboard: Display a summary of the questionnaire results, including the total score, category, priority, score breakdown (bar chart), a priority matrix, key insights, and a red flags section.
- Database integration: Store the response and calculate scores of each questionnaire

## Style Guidelines:

- Primary color: A vibrant azure blue (#4285F4) evoking a sense of trust, intelligence, and efficiency.
- Background color: A light, desaturated cyan (#E3F2FD) providing a clean and calming backdrop.
- Accent color: A bright spring green (#B0E57C) to indicate positive metrics and actionable items.
- Headline font: 'Space Grotesk' (sans-serif) for a techy and modern aesthetic, making titles and important information stand out.
- Body font: 'Inter' (sans-serif) providing a clean and readable experience for forms, descriptions and detailed insights.
- Use simple, geometric icons in the primary color (#4285F4) to visually represent data categories and questionnaire sections, ensuring clarity and ease of navigation.
- Implement a clear, structured layout with logical grouping of questions, progress indicators, and easily accessible help text. The goal is a user-friendly experience that encourages accurate and complete responses.
- Use subtle animations for transitions, loading states, and score updates to enhance user engagement and provide a smooth, professional experience.
